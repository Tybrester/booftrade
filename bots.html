TradingView Setup Guide

Step 1 — Add this to your Pine Script strategy

// Add these lines to your Pine Script strategy // Replace "buy_signal" and "sell_signal" with your actual conditions alert(str.format('{"action":"buy","symbol":"{0}","price":{1},"quantity":1,"order_type":"market","broker":"tastytrade"}', syminfo.ticker, close), alert.freq_once_per_bar_close) alert(str.format('{"action":"sell","symbol":"{0}","price":{1},"quantity":1,"order_type":"market","broker":"tastytrade"}', syminfo.ticker, close), alert.freq_once_per_bar_close) // OR use alertcondition() for strategy.entry signals: alertcondition(strategy.position_size > 0, "Buy Signal", '{"action":"buy","symbol":"{{ticker}}","price":{{close}},"quantity":1,"broker":"tastytrade"}') alertcondition(strategy.position_size < 0, "Sell Signal", '{"action":"sell","symbol":"{{ticker}}","price":{{close}},"quantity":1,"broker":"tastytrade"}')

Step 2 — Create a TradingView Alert

1. On your chart, right-click → Add Alert
2. Set condition to your Pine Script signal
3. Under Notifications, enable Webhook URL
4. Paste your bot's Webhook URL (shown on each bot card above)
5. Set the Message to the JSON payload from Step 1
6. Enable the bot using the toggle above — done!
